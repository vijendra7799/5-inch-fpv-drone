# 🚁 5-Inch FPV Freestyle Drone

> A custom-built 5-inch FPV freestyle drone designed, assembled, configured, and flight-tested from individual components.

![Drone](images/drone.jpg)

---

## 📖 Overview

This project involved building a complete 5-inch FPV freestyle drone from scratch, including hardware assembly, electronics integration, firmware configuration, and flight testing.

The drone was built to gain practical experience in UAV systems, embedded electronics, flight controller configuration, and troubleshooting. The entire system was configured using Betaflight and ExpressLRS, followed by PID tuning and real-world flight testing.

---

## 🎯 Objectives

- Build a fully functional FPV freestyle drone
- Learn drone electronics and hardware integration
- Configure and tune a flight controller
- Implement reliable radio communication using ExpressLRS
- Integrate GPS for navigation and rescue features
- Perform safe flight testing and troubleshooting

# 🛠 Components Used

| Component | Model |
|-----------|-------|
| Frame | *Mark4 7inch 295mm Carbon Fiber Frame Kit* |
| Flight Controller | SpeedyBee F405 |
| ESC | *Speedy Bee Stack* |
| Motors | *Readytosky RS2205 2300KV CW Brushless Motor* |
| Propellers | 5-inch Tri-Blade |
| Receiver | ExpressLRS Nano Receiver |
| Radio Transmitter | Radiomaster Pocket |
| GPS Module | HGLRC M100 GPS |
| FPV Camera | *1500TVL FPV Camera* |
| Video Transmitter | SpeedyBee TX800 |
| Antenna | MMCX Antenna |
| Battery | 4S Li-ion 2800 mAh |

# 🔌 Hardware Assembly

The drone was assembled by integrating all electronic components onto a carbon fiber frame.

Major assembly tasks included:

- Frame assembly
- Motor installation
- ESC installation
- Flight controller mounting
- Receiver wiring
- GPS installation
- FPV camera installation
- VTX installation
- Battery mounting
- Propeller installation

Special attention was given to clean cable management and vibration isolation for stable flight performance.

---

# ⚡ Wiring

The following components were connected to the SpeedyBee F405 Flight Controller:

- ESC
- ExpressLRS Receiver
- GPS Module
- FPV Camera
- Video Transmitter
- Buzzer (if applicable)

Proper UART configuration was performed to enable communication between peripherals.

---

# 💻 Software Configuration

The drone was configured using:

- Betaflight Configurator
- ExpressLRS Configurator
- SpeedyBee Mobile App

Configuration steps included:

- Flashing firmware
- Configuring UART ports
- Receiver setup (CRSF)
- OSD configuration
- Failsafe setup
- GPS configuration
- Arming configuration
- Flight modes
- Motor direction verification

---

# 📡 ExpressLRS Configuration

The radio communication system was configured using ExpressLRS.

Tasks completed:

- Receiver flashing
- Binding with transmitter
- Packet rate verification
- Telemetry setup
- RSSI monitoring
- Failsafe configuration

---

# 🛰 GPS Integration

The HGLRC M100 GPS module was integrated with the flight controller.

Features enabled:

- Satellite lock
- GPS telemetry
- Home point detection
- GPS Rescue

---

# 🎥 FPV System

The FPV system consists of:

- Analog FPV Camera
- SpeedyBee TX800 Video Transmitter
- MMCX Antenna

The video transmitter power levels and channels were configured using Betaflight.

# 🎛 PID Tuning

After the initial setup, PID tuning was performed to improve flight characteristics.

Tuning objectives:

- Stable hover
- Reduced oscillations
- Smooth cornering
- Improved freestyle response

Multiple test flights were conducted before finalizing the tuning parameters.
